Bubble API

フリーミアム グローバル ⚡ NoCode・自動化 中級

ビジュアルWebアプリ開発プラットフォームのAPI。Data API（データ操作）とWorkflow API（ワークフロー実行）を提供し、Bubbleで構築したアプリのバックエンドを外部から操作できる。ノーコードでフルスタックWebアプリを構築可能

人気スコア（100点満点）

詳細分析

BubbleはノーコードWebアプリ開発プラットフォームのリーダー。Data APIとWorkflow APIで外部連携が可能。ワークロードユニット（WU）制の料金体系で、API操作もWUを消費。無料プランは開発モードのみで公開にはStarterプラン以上が必要。
