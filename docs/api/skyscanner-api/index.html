Skyscanner API

フリーミアム グローバル ✈️ 旅行・交通 中級

世界最大級の旅行メタサーチエンジン「Skyscanner」のAPI。航空券・ホテル・レンタカーの価格検索データを提供し、Flights Live Prices API（リアルタイム料金）、Flights Indicative Prices API（概算料金）、Car Hire API、Hotels APIを利用可能。パートナープログラムを通じてアクセス

75
人気スコア（100点満点）
世界100+市場で展開する旅行メタサーチの大手。パートナーAPIを通じて多数の旅行サイト・アプリにデータを提供

詳細分析

Skyscannerは2003年にスコットランドで創業し、現在はTrip.comグループ傘下。世界100以上の市場で航空券・ホテル・レンタカーの比較検索を提供。APIはパートナープログラムを通じて利用可能。
