Backblaze B2 API

Backblaze B2 クラウドストレージAPI
フリーミアム グローバル ☁️ ストレージ 中級

S3互換の低コストクラウドオブジェクトストレージ「Backblaze B2」のAPI。S3互換APIとB2ネイティブAPIの2種類を提供し、認証、バケット管理、ファイルのアップロード・ダウンロード、メタデータ操作などをサポート。AWS S3の約1/5のコストで利用でき、帯域幅料金も無料。アップロードAPIコールは課金対象外で、大容量データの保存・配信に最適。

70
人気スコア（100点満点）
AWS S3の約1/5のコストと帯域幅無料が最大の差別化要因。S3互換APIで既存アプリからの移行が容易

詳細分析

Backblaze B2はAWS S3の低コスト代替として人気。帯域幅無料・アップロードAPI無料の料金体系で、特にバックアップ・アーカイブ用途での採用が多い。Cloudflare CDNとの無料連携（Bandwidth Alliance）も魅力。
