APIpedia について

APIpedia は、日本の開発者が言語の壁なくAPIを発見・理解・活用できることを目指す日本語APIカタログです。統一フォーマットでAPIの概要・料金・制限を整理し、「探す」を「使う」に最短距離でつなぎます。
